Historia del HTML

Origen de este:
  • Se creo este formato en el año de 1980 por Tim Berners trabajador de la CERN penso usar un nuevo tipo de hiper vinculo para el sitio web oficial para la entidad pero en su tiempo habian otros que no eran efectivas en la practica.

  • Tras finalizar el desarrollo de su sistema de "hipertexto", Tim Berners-Lee lo presentó a una convocatoria organizada para desarrollar un sistema de "hipertexto" para Internet. Después de unir sus fuerzas con el ingeniero de sistemas Robert Cailliau, presentaron la propuesta ganadora llamada WorldWideWeb.

  • En 1995, el organismo IETF organiza un grupo de trabajo de HTML y consigue publicar, el 22 de septiembre de ese mismo año, el estándar HTML 2.0. A pesar de su nombre, HTML 2.0 es el primer estándar oficial de HTML.

    Caracteristicas de este metodo:
  • Se caracteriza por ser algo mas sencillo para principantes en esta materia importante de la informatica, requiere solo entender el ingles tecnico basico y ser ordenado en tu codigo y por supuesto un equipo de computacion.

  • Puede ser mezclado con otros sistemas de programacion como PYTHON, CSS, JS (En su mayoria se uan estos para sitios web)

    Sus versiones a traves de los años:
  • Fue la version que trajo todo lo basico de los posteriores como los comandos, argumentos y fue la version que se basaron para crear su segunda version que fue la por defecto en los años 1995, hasta que se introducio la version 3.0 y en 1997 la version 3.2, y en 1999 se introducio la version 4.0, la version que se uso hasta la introduccion de la version actual HTML 5 que aun se usa constantemente como base de los websites modernos.
